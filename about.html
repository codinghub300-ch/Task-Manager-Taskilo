Introduction

Task Manager is a comprehensive web application designed to boost your productivity by helping you manage, track, and visualize your tasks with ease. With features such as drag & drop organization, priority settings, dark/light mode, and dynamic charts, managing your daily tasks has never been more efficient.

Features

Task Management
  • ✅ Add, update, and remove tasks
  • 🔄 Drag & drop between categories
  • 📅 Set due dates and reminders
Visual Analytics
  • 📊 Gold-themed charts
  • 📈 Real-time task progress
  • 🎯 Overall golden frames
Customization
  • 🌙 Dark/Light mode toggle
  • 🔖 Task priority & labels
  • 💬 Collaborative features

How to Use

Get Started
  • Type in your task
  • Set due dates & priorities
  • Click "Add Task"
Organize Tasks
  • Drag tasks to re-order
  • Move tasks between categories
  • Use charts to track progress
